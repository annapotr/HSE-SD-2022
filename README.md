## SShell-like CLI interpreter
### Сборка
- на MacOS или Linux:

  `./gradlew SShellJar`
- на Windows:

  `.\gradlew.bat SShellJar`

### Запуск интерпретатора
`java -jar build/libs/SD-2022.jar`

### Запуск тестов
- на MacOS или Linux:

  `./gradlew test`
- на Windows:

  `.\gradlew.bat test`
