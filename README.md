## SShell CLI interpreter
### Сборка
- на MacOS или Linux:

  `./gradlew SShellFatJar`
- на Windows:

  `.\gradlew.bat SShellFatJar`

### Запуск интерпретатора
`java -jar build/libs/SD-2022.jar`

### Запуск тестов
- на MacOS или Linux:

  `./gradlew test`
- на Windows:

  `.\gradlew.bat test`
